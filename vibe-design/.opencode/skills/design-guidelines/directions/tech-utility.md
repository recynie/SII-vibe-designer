# Direction · Tech Utility

参考：Datadog, GitHub, Cloudflare dashboard, Sentry

## 氛围

数据密集、等宽友好、网格感。为工程师和运维人员设计——单位面积信息量优先，不追求氛围感。

## 字体

- Display: -apple-system, BlinkMacSystemFont, 'Inter', 'Segoe UI', system-ui, sans-serif
- Body: -apple-system, BlinkMacSystemFont, 'Inter', 'Segoe UI', system-ui, sans-serif
- Mono: 'JetBrains Mono', 'IBM Plex Mono', ui-monospace, Menlo, monospace

## Posture

- 无衬线做 display + body（同一字族）可以接受——工具性优先于编辑性
- 全局等宽数字，代码/ID/hash 用 mono
- 紧凑表格 + hairline 边框，不做行条纹
- 状态标签（success/warn/danger）用克制的浅色背景
- 避免：hero 大图、超大标题、营销文案——展示产品本身
