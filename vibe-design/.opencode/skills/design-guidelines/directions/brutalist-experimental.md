# Direction · Brutalist Experimental

参考：Are.na, Yale Center for British Art, mschf, Read.cv

## 氛围

大字体。可见网格。系统无衬线 + 单个超大衬线。刻意的粗糙即自信。适合艺术、独立、agency、宣言页面。

## 色板

| 角色 | hex | 说明 |
|---|---|---|
| Background | #F8FAFD | 中性打印纸 |
| Surface | #FFFFFF | 纯白 |
| Ink | #1A1A1A | 近黑 |
| Muted | #525252 | 深灰 |
| Border | #1A1A1A | 边框 = 前景色全强度 |
| Accent | #DC2626 | 热红 |

## 字体

- Display: 'Times New Roman', 'Iowan Old Style', Georgia, serif
- Body: ui-monospace, 'IBM Plex Mono', 'JetBrains Mono', Menlo, monospace

## Posture

- Display = 衬线极大尺寸 (clamp(80px, 12vw, 200px))
- Body = 等宽——是的，等宽做正文，刻意为之
- 边框全强度前景色（1.5-2px），不用灰色
- 非对称布局：一列 70%，另一列 30%
- 几乎不用圆角（0-2px）。不用阴影。不用渐变。
- 下划线链接，不加 hover 装饰——让字排自己说话
