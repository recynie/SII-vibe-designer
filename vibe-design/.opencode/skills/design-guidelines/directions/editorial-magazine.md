# Direction · Editorial Magazine

参考：Monocle, The Financial Times Weekend, NYT Magazine, It's Nice That

## 氛围

印刷杂志感。适用于明确的编辑/出版 brief。大面积留白、大号衬线标题、克制的色彩参考（中性纸色 + 墨色 + 单一品牌色强调）。不要用于商业 SaaS、仪表盘或产品工具页面。

## 色彩参考

| 角色 | 参考 | 说明 |
|---|---|---|
| Background | #F8F5F0 | 中性纸色（不是米黄） |
| Surface | #FFFDF9 | 近白暖调 |
| Ink | #2C2518 | 墨色 |
| Muted | #6B6154 | 中灰暖 |
| Border | #E5E0D8 | 浅灰暖 |
| Accent | #B03A2E | 克制的编辑红 |

## 字体

- Display: 'Iowan Old Style', 'Charter', Georgia, serif
- Body: -apple-system, BlinkMacSystemFont, 'Segoe UI', system-ui, sans-serif

## Posture

- 衬线做 display，无衬线做 body，等宽仅用于 metadata
- 不用阴影、不用圆角卡片——靠边框 + 留白建立结构
- 一张决定性的图片，仅从底部裁切
- kicker/eyebrow 用等宽大写，强调色最多出现两次；除非品牌要求，不做蜜桃/粉色/橙米色的页面底色
