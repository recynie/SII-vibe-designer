# Skill · Copywriting（介质工艺手册）

文案介质：纯 markdown。**不调 gen_image，不写 HTML**。

## 工艺要点
- Slogan ≤ 14 字（中）/ 8 词（英），3 个变体并列；每条标定位（学术/进取/通用 等）
- 一段简介 80-120 字，第三人称，**不出现"我们"**
- 中文标点用「」不用 ""

## 反 slop 红线
- 词："赋能 / 打造 / 生态 / 全链路 / 垂直 / 一站式"
- 强行押韵 / 强行对仗 / 感叹号收尾
- 英文版逐字翻译中文
- "世界 / 未来"等空词，除非 brief 要

## 正向
- 具体的、可被反驳的句子（"做 A 不做 B"）
- 给读者一个画面或动作
