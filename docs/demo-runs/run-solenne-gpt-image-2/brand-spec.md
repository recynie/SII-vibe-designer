# Brand Spec

> 采集日期：2026-05-16
> 来源：用户 brief + 独立香水行业趋势推断

## 色板

| 角色 | 色值 | 说明 |
|------|------|------|
| Primary | #1A2F1A | 暗森林绿，呼应"暗夜植物"调性 |
| Secondary | #3D3D3D | 深炭灰，用于辅助元素 |
| Background | #F5F2EB | 象牙白/米纸色，还原纸质触感 |
| Ink | #1A1814 | 近黑深棕，正文用色 |
| Accent | #C9A961 | 古铜金，点缀（标签/线条） |

（禁用紫渐变，除非 brief 明确要求）

## 字体方向

- **Display（标题/wordmark）**：衬线体，推荐 **Cormorant Garamond** 或 **Playfair Display**——有 editorial 庄严感，与 Solenne（意为"庄严的"）名字含义呼应
- **Body（正文）**：极简无衬线，推荐 **system-ui** / **San Francisco** —— 避免授权问题，保持干净

## 视觉气质

3-5 个形容词，与调性关键词一致：
- 暗夜森林感（深绿主调）
- 克制的奢华（不过度装饰）
- 留白呼吸（大量负空间）
- 纸质触感（可选纹理）
- 叶脉/脉络线条（可选图形元素）

## 设计参考方向

- **方向**：Pentagram + 东方极简融合
- **理由**：Pentagram 的信息建筑派适合克制精简的品牌；加入东方极简的留白哲学，呼应"嗅觉至上"——视觉不抢嗅觉的风头
- **不适合**：过度艺术化的解构主义、独立设计师的怪诞风格

## 反 AI slop 禁区（针对本项目）

- 不要：紫渐变、emoji 图标、generic stock photo
- 不要：Inter 当 display 字体、圆角卡通 logo
- 不要：甜腻粉色系、渐变玻璃质感
- 避免：过多颜色（色板上限 5 个）

---

> 注：此规格为最小验证版设计基准，设计师可在此范围内探索三变体 logo 与 slogan 组合。
