## 改进建议

### 1. [prompt 问题] 颜色精度未强调
**根因**：prompt 中写 "dark forest green #1A2F1A"，但 AI 生成的 #100100e0 偏离了绿相。

**建议**：
- 在 prompt 中强化颜色约束：`primary color MUST be exact #1A2F1A (RGB 26,47,26) dark forest green, NOT purple, NOT blue-black`
- 或增加负面约束：`avoid any purple or blue tint in dark areas`
- 考虑在 prompt 开头直接指定色板而不只是描述

### 2. [素材问题] 生成的图形色相不可控
**根因**：图像生成模型对"dark forest green"的理解可能偏向更深的颜色，导致生成偏紫的暗色。

**建议**：
- 改用更具体的描述：`deep hunter green` 或 `British racing green`
- 或者：考虑用 SVG 代码直接指定颜色，而非依赖图像生成

### 3. [版式问题] 无（版式已合理）
三变体水平排列清晰，各区域分隔合理，无需调整。
