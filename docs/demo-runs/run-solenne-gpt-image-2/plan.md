# Plan — Solenne 最小验证版

## 任务清单

| 序号 | 任务 | skill | 预期输出 |
|------|------|-------|----------|
| 1 | Logo 三变体 | logo | artifacts/logo/v1.png (mark + wordmark + combo) |
| 2 | Slogan × 3 | copywriting | artifacts/copy/v1.txt |

## 执行规则
- 每个任务只迭代 1 轮
- Logo：一次性出 mark + wordmark + combo 三变体
- Slogan：一次性出 3 条
- 串行执行：先 logo → 评审 → slogan → 评审 → 汇总

## 验收标准
- Logo：三变体完整，气质克制、暗夜植物感
- Slogan：3 条，每条体现"嗅觉至上"或"克制"调性
