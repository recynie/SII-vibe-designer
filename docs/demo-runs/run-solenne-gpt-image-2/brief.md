# Design Brief

## 主题
Solenne 独立香水工作室

## 用户原话
> 为一个叫 'Solenne' 的独立香水工作室做品牌设计的最小验证版：只做 1 个 logo（mark + wordmark + combo 三变体）+ 3 条 slogan，每个任务只迭代一轮，最快路径产出 + 汇总。气质：克制、嗅觉至上、暗夜植物香。

## 核心信息
- 是什么：独立香水工作室，专注于小众香氛创作
- 做什么：以植物为核心原料的定制香水，强调嗅觉体验的纯粹性
- 给谁看：追求小众、质感、内敛的香水爱好者；愿意为独特气味支付溢价的品味消费者

## 调性关键词（5 个）
- 克制 / 嗅觉至上 / 暗夜植物 / 庄严 / 内敛

## 必含元素
- Logo 三变体：纯图形 mark、纯字 wordmark、组合 combo
- Slogan × 3 条

## 禁含元素
- 不要：紫渐变、emoji 图标、generic stock photo
- 不要：过度装饰、甜腻感、流行快消视觉

## 交付清单
- [ ] Logo 主标志（三变体）
- [ ] Slogan × 3 条
- [ ] 品牌文案（slogan + 一段简介）
