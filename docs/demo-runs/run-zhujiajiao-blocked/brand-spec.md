# Brand Spec

> 采集日期：2026-05-16
> 来源：WebSearch / 维基百科朱家角词条、青浦区政府官网

## 色板
- Primary: #1A2634 「黛青」——呼应古镇水墨意境，呼应"朱"字的历史沉淀
- Secondary: #C41E3A 「朱砂」——"朱家角"姓氏标识，传统中国色，点睛之笔
- Background: #F5F2EB 「宣纸白」——仿宣纸质感的暖白，克制纯白
- Ink (正文): #2D2D2D 「墨色」——接近黑色的深灰，比纯黑柔和
- Accent: #4A7C6F 「竹青」——江南园林与竹元素的自然呼应

（不超过 5 个，禁用紫渐变除非 brief 明确要求）

## 字体方向
- Display: 思源宋体（Noto Serif SC）Regular/Bold —— 中文衬线首选，兼具宋刻本气质与现代可读性；备选：方正书宋、筑紫明朝
- Body: 思源黑体（Noto Sans SC）—— 无衬线安全牌，避免授权风险；备选：OPPOSans、站酷高端黑（需确认授权）

## 视觉气质
3-5 个形容词，与"调性关键词"一致但描述视觉表现：
- 水墨留白：大面积负空间，避免信息过载
- 当代极简：去除传统装饰元素，用几何线条抽象古镇符号
- 江南雅致：低饱和配色，克制张狂
- 轻盈现代：字体排版偏细偏紧，不厚重

## 设计参考方向
- 吕敬人「书籍建筑」派信息整理术——将古镇历史脉络以现代信息层级呈现
- 原研哉「无印良品」美学——白、留白、素材本位、无品牌胜似品牌
- 说明：二者结合，打造"当代东方水乡"的视觉语言——不是复古拷贝，而是用当代设计语言重新诠释江南意境

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变、emoji 图标、generic stock 图库、Inter 当 display 字体
- 不要：传统红灯笼、中国结等刻板古镇元素
- 不要：高饱和度渐变色、大面积高亮色块
- 不要：过度装饰的云纹/水纹/祥云纹样
