# Plan - 朱家角古镇文旅品牌设计

## 任务清单

| # | 子任务 | Skill | 输出文件 | 备注 |
|---|--------|-------|----------|------|
| 1 | Logo 主标志 | logo | artifacts/logo/v1.png + v1.prompt.txt | 需适配多场景 |
| 2 | 宣传海报主视觉 | poster | artifacts/poster/v1.html + v1.png | 静态展示 |
| 3 | 文创产品命名 + 文案 | copywriting | artifacts/copy/v1.md | 一组产品线 |
| 4 | 旅游导览 H5 首屏 | ui-mockup | artifacts/ui-mockup/v1.html | 移动端适配 |

## 执行顺序

严格串行：1 → 2 → 3 → 4，每轮由 designer 执行 → critic 评审（最多 2 轮）。

## 验收标准

- Logo：现代水墨风，适配数字/印刷/导视
- 海报：视觉冲击力 + 江南气质
- 文创：年轻化命名 + 打动 25-35 岁
- H5：移动端首屏可用，交互合理
