# Design Brief

## 主题
上海朱家角古镇文旅品牌形象设计

## 用户原话
> 请为上海朱家角古镇做一套文旅品牌形象设计。朱家角是江南水乡六大古镇之一，明清建筑、河道纵横、桥梁林立。这次设计目标是面向年轻一代游客（25-35）做一次品牌焕新，既要保留古镇的水墨气质，又不能太刻板老气。交付物至少包含：文旅 logo、宣传海报主视觉、文创产品命名 + 文案、旅游导览 H5 移动端首屏。

## 核心信息
- 是什么：朱家角是上海青浦区的江南水乡古镇，1991年被评为上海市第一批历史文化名镇，旧称珠溪镇、珠街阁，俗称角里，明万历年间正式建镇
- 做什么：面向25-35岁年轻游客的文旅品牌焕新，保留水墨气质但去除刻板老气
- 给谁看：25-35岁年轻一代游客，追求文化体验与生活方式消费，注重审美品质和在地文化深度

## 调性关键词（5 个）
- 水墨 / 雅致 / 当代 / 江南 / 焕新

## 必含元素
- [ ] Logo 主标志
- [ ] 宣传海报主视觉
- [ ] 文创产品命名 + 文案
- [ ] 旅游导览 H5 移动端首屏

## 禁含元素
- 避免传统古镇旅游的刻板印象：红灯笼、假古董、义乌小商品感
- 禁止紫渐变、generic stock 图库、emoji 图标
- 避免过于厚重传统的书法字体

## 交付清单
- [ ] 文旅 logo（需适配多种场景：导视、周边、数字媒体）
- [ ] 宣传海报主视觉一件
- [ ] 文创产品命名 + 包装文案（一组产品线）
- [ ] 旅游导览 H5 移动端首屏 mockup
