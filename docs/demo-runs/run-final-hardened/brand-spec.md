# Brand Spec

> 采集日期：2026-05-16
> 来源：用户 brief 推断（无公开信息）

## 色板

- Primary: #0D1B2A（深海蓝——学术稳重，信任感，契合研究机构定位）
- Secondary: #1B4965（青墨蓝——科技感与专业性的平衡）
- Background: #F8F9FA（冷白——洁净、理性、适合学术阅读）
- Ink: #1A1A2E（墨色——正文与核心文字，高对比度确保可读性）
- Accent: #00B4D8（青空蓝——点缀色，代表 AI 与未来感，克制使用）

## 字体方向

- Display（标题/标志）：Noto Serif SC（思源宋体）或 Source Han Serif CN，衬线体现学术沉淀感，权重建议使用 Bold/Heavy。
- Body（正文）：Noto Sans SC（思源黑体）或 system-ui，无衬线确保屏幕阅读舒适性，权重 Regular/Medium。
- 备选 Display：Fandol Song（开源宋体，配合开源精神）
- 备选 Body：LXGW WenKai（开源字体，体现自主研发气质）

## 视觉气质

- 高对比：深色文字配浅色底，清晰传达信息
- 几何精简：使用基础几何形（矩形、圆形、三角形），避免装饰冗余
- 留白克制：大量留白体现学术机构的呼吸感与从容
- 科技冷感：避免暖色调，主调偏冷蓝，辅以青色点缀

## 设计参考方向

- **Pentagram 信息建筑派**：强调信息的层级、结构与功能，契合研究机构理性、严谨的特质。以网格系统驱动设计，元素排列有序。
- 备选参考：**原研哉（Kenya Hara）东方极简**：重视「白」的意境，赋予品牌文化厚度与从容感，契合上海国际化都市背景下的学术机构定位。

## 反 AI slop 禁区（针对本项目）

- 禁止使用紫渐变或粉紫系配色（与学术稳重调性冲突）
- 禁止使用 emoji 或表情图标（降低专业感）
- 禁止使用 generic stock 照片（如握手、团队开会等模板图）
- 禁止将 Inter 字体用作 Display（Inter 为正文屏显字体，缺乏标识性）
- 禁止使用发光/辉光效果过度装饰（保持克制、理性的视觉基调）
