# Design Brief

## 主题

创智学院

## 用户原话

> 创智学院是上海一所专注于人工智能创新与人才培养的研究型机构，定位偏向研究院 + 高等教育混合体，面向青年研究者和工程师，希望气质上既有学术稳重感、又有面向未来的现代感。交付物至少包含：logo 主标志、品牌主视觉海报、品牌文案（slogan + 简介）、官网首页 mockup。

## 核心信息

- 是什么：创智学院是一所位于上海的人工智能研究型机构，兼具研究院与高等教育双重属性。
- 做什么：聚焦人工智能领域的创新研究与人才培养，培养青年研究者和工程师。
- 给谁看：青年研究者和工程师、潜在学员、学术界同行、产业合作伙伴。

## 调性关键词（5 个）

- 学术
- 稳重
- 现代
- 创新
- 前沿

## 必含元素

- [ ] Logo 主标志
- [ ] 品牌主视觉海报
- [ ] 品牌文案（slogan + 简介）
- [ ] 官网首页 mockup

## 禁含元素

- 避免过度娱乐化或萌系风格
- 避免使用低龄化或过于活泼的视觉元素
- 避免与大众互联网产品混淆的设计语言

## 交付清单

- Logo 主标志
- 品牌主视觉海报
- 品牌文案（slogan + 简介）
- 官网首页 mockup（一页）
