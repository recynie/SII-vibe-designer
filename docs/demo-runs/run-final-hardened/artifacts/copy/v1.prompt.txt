# v1 创作思路注解

## 调性把控

- **学术稳重**：通过「研究型机构」「前沿领域」「智力支持」等词体现，避免轻浮表达
- **现代前沿**：通过「AI 时代」「定义问题」「产业前线」等词体现，避免陈旧表述
- **面向青年**：用「未来的研究者」「同行者」「解开问题」等语气，平等而非说教

## 词汇选择纪律

- ❌ 禁用「赋能」「打造」「生态」「垂直」「一站式」
- ✅ 选用具体动词：「深研」「致用」「解开」「定义」「架通」
- ✅ 选用具体名词：「问题」「课题」「研究」「视野」「同行者」

## Slogan 策略

- A「智以深研，创以致用」：对仗工整，呼应「创智」名字，体现学术+创新双重基因
- B「在 AI 时代，成为定义问题的人」：进取感强，面向有野心的青年研究者
- C「学问生长于实验室，也生长于产业前线」：稳健，呼应「研究院+高等教育」双重属性

## 简介策略

- 第三人称叙述，不出现「我们」
- 突出「上海」「研究型」「双重属性」三个核心标签
- 落脚点在「人才支撑与智力支持」，务实不虚

## 应用文案选择

- 欢迎信：面向潜在学员，语气平等但有学院底气
- 合作邀约：面向企业/高校，列点清晰，体现专业感
