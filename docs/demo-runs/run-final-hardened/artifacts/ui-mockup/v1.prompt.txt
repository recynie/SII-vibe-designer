# UI Mockup 设计说明

## 设计方向
- 学术稳重 × 现代前沿
- 网格系统驱动设计，参考 Pentagram 信息建筑派
- 重视信息层级、结构与功能，契合研究机构理性、严谨的特质

## 色板（严格遵守 brand-spec.md）
- Primary: #0D1B2A（深海蓝）——导航CTA、研究方向标题、招生区背景
- Secondary: #1B4965（青墨蓝）——卡片渐变、头像区背景
- Accent: #00B4D8（青空蓝）——标签、链接hover、强调元素
- Background: #F8F9FA（冷白）——页面主背景
- Ink: #1A1A2E（墨色）——正文文字

## 字体配对
- Display（标题）: Noto Serif SC（思源宋体）700/900
- Body（正文）: Noto Sans SC（思源黑体）400/500

## 板块结构
1. **导航栏** - logo + 4个链接 + CTA按钮，sticky定位
2. **Hero区** - 主标题 + 副标题 + CTA + 抽象几何视觉（AI字母图案）
3. **研究方向** - 3栏卡片布局，每栏标题 + 描述 + 标签
4. **师资团队** - 4栏卡片，头像区用渐变色块+姓氏占位
5. **招生培养** - 3栏卡片，深色背景突出
6. **页脚** - logo + 版权 + 链接

## 反 AI slop 措施
- ✅ 没用圆角卡片 + 左侧 accent border（避免SaaS模板风）
- ✅ 没用 emoji 图标（使用简洁文字标签）
- ✅ 没用紫渐变（严格用蓝青色系）
- ✅ 没用 placeholder.com 灰色占位（用代码生成的视觉元素）
- ✅ 没用"赋能"、"一站式"等套话
- ✅ 没把 hero 配图做成 SVG 几何拼贴（用抽象色块+字母）

## 移动端适配
- 断点 1024px：hero 改为单栏，研究/师资 改为2栏
- 断点 768px：隐藏导航链接，改为极简布局，所有区块单栏
