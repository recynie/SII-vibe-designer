# Deliverables · 上海朱家角品牌形象设计

## 显式

- 方案一：年轻化传播视觉方案 | 独立品牌视觉方案，面向年轻游客和社媒传播，整体以现代水乡、轻快打卡、APP 友好为重点，不与方案二共用同一套标志系统
  - Logo / 标志 | 1024×1024 PNG 效果图，扁平矢量风，围绕朱家角水网、放生桥、角里字形或摇橹船形成年轻化组合标志
  - 主视觉海报 | 1200×1600 HTML 海报 + 渲染 PNG，竖版文旅推广海报，体现“小桥、流水、人家”、古镇焕新和周末游吸引力
  - 社媒传播图 | 1080×1080 HTML 社交媒体方图 + 渲染 PNG，适合小红书/朋友圈封面，突出“最江南”“水乡打卡”“年轻周末游”信息层级
  - 宣传册 | A4 三折页展开版 2480×1754 HTML 宣传册 + 渲染 PNG，包含核心景点、游船、北大街、课植园与推荐动线信息
  - 旅游 APP 界面 | 390×844 HTML 移动端首屏 + 渲染 PNG，呈现朱家角旅游 APP 首页概念，包括景点卡片、路线、票务/导览入口
- 方案二：高端文化视觉方案 | 独立品牌视觉方案，面向文化旅游、高品质导览与公共空间应用，整体以历史文脉、园林秩序、水墨留白为重点，不与方案一共用同一套标志系统
  - Logo / 标志 | 1024×1024 PNG 效果图，克制徽章或字标方向，围绕放生桥五孔、课植园、珠溪/角里文脉或水纹负空间形成高端文化标志
  - 主视觉海报 | 1200×1600 HTML 海报 + 渲染 PNG，竖版文化推广海报，强调千年古镇、明清街巷、江南文脉与“最江南”气质
  - 导视或文旅应用物料 | 1024×1024 PNG 公共空间导视效果图，呈现古镇入口或滨水节点导览牌/方向牌，强调石材、木作、黛瓦与可读中文信息
  - 明信片 | 1800×1200 HTML 明信片正反面展示图 + 渲染 PNG，围绕放生桥、北大街、课植园或漕港河形成高端纪念品版式
  - 公共家具设计 | 1024×1024 PNG 公共家具效果图，呈现导览牌、休憩牌或滨水导览座椅等文旅公共家具，强调与古镇街巷和水岸环境融合

## 隐式

- 双方案品牌提案总览 | 1440×1800 HTML 提案板 + 渲染 PNG，对比两套独立方案的核心意象、色彩、字体与应用侧重；推断理由：用户明确要求两套独立视觉方案，需要总览页帮助评审区分两套系统

## 拒绝

- 短视频 / 动效宣传片 | brief 未要求视频媒介，且超出本 run 平面与静态视觉设计范围
- 完整导视施工图与公共家具结构图 | 本 run 只做品牌视觉与效果图，不做工程落地尺寸、材料清单或施工节点
- 全量品牌手册 PDF | 超出 5 件上限，本 run 以两套视觉方案及关键应用效果为主，不制作完整规范手册
- 官方素材强制沿用 | 调研下载的官网 logo 与图片仅作现状参考，用户要求完成新的品牌形象设计，未要求沿用官方既有 logo 或照片

## 决策依据

- 显式条目来自 brief「需要两套独立视觉方案」「方案一：年轻化传播方向，需包含 logo / 标志、主视觉海报、社媒传播图、文创延展」「方案二：高端文化方向，需包含 logo / 标志、主视觉海报、导视或文旅应用物料、文创延展」。
- 方案一子项中的宣传册、旅游 APP 界面来自 brief「文创延展具体侧重：年轻化方案做宣传册、旅游 APP 界面」。
- 方案二子项中的明信片、公共家具设计来自 brief「高端文化方案做明信片、公共家具设计（导览牌等）」。
- 核心意象依据 facts.md「区内河港纵横成网，九条百年老街沿河蜿蜒，千栋明清建筑依水亭立，36 座古桥古风犹存」「放生桥」「北大街」「课植园」「小桥、流水、人家」。
- 两个主条目使用 sub-items，是因为每套方案包含多个独立文件且共享同一套视觉逻辑；同时用户强调两套方案独立，因此分为两个主交付物而不是同一系统的变体。
- 隐式总览页依据用户要求“两套独立视觉方案”，用于明确展示两套方案在意象、色彩、字体和应用侧重上的差异，避免 designer 产出同质化换色方案。
- 拒绝项依据本 run 平面/静态视觉范围与显式+隐式主条目合计 3 条，未超 5 件上限。
