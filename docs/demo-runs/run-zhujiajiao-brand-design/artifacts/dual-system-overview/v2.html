双方案品牌提案总览

同一朱家角语境下的两套独立品牌系统：左侧以清透界面、移动端传播和年轻周末游为主；右侧以纸张肌理、水墨留白和公共文化空间为主。结构、字体、容器语言均分开表达。
overview board · 1440×1800
不再是同模板换色，
而是两种观看方式。

方案一像一组可操作的旅游界面；方案二像一套可收藏、可落地的文化纸本与空间系统。

方案一 · 年轻化传播视觉系统

清水青绿与银杏黄组成轻快、透明、可分享的界面语言，重点服务社媒封面、APP 首页、宣传册和周末游转化。

Brochure + App 形成移动端闭环 宣传册不伪造新内容，直接使用已生成三折页输出作为微缩预览；与 APP 首页共同补全“线下拿取—线上导览”的年轻传播路径。

core imagery

水网、放生桥、打卡路径、古镇焕新；强调“可参与的江南”，阅读速度快，容器更接近屏幕组件。
