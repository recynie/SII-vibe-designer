Shanghai Zhujiajiao Brand Proposal

双方案品牌提案总览

同一古镇语境下的两套独立系统：左侧以年轻化传播和轻快打卡为核心，右侧以历史文脉、园林秩序与文博级气质为核心。以下总览从核心意象、色彩、字体与应用侧重四个维度直接对照。
overview board · 1440×1800
朱家角双方案，不同系统，不同气质。

避免同一 logo 换色化处理，强调真正独立的视觉语言与应用场景分化。

方案一 · 年轻化传播视觉系统

清水青绿与银杏黄构成高识别度的水感传播界面，面向年轻游客、小红书/朋友圈扩散与旅游 APP 场景，强调轻盈、现代、可分享。
modern / lively / shareable
core visual poster
核心意象
水网、放生桥、周末游、打卡路径、古镇焕新；视觉重点落在“可参与的江南”，更像一套面向社媒传播的旅游内容系统。
色彩
清水青绿 湖水青 银杏黄 宣纸白
字体
Display / Body
Noto Sans SC · Heavy / Regular
应用侧重
Logo、主视觉海报、社媒传播图、宣传册、旅游 APP 首页概念。整体偏轻、偏快、偏移动端阅读。

方案二 · 高端文化视觉系统

黛瓦墨青与米金暖灰建立更克制的文脉秩序，以历史厚度、留白和材质感呈现朱家角的文化深度，适配导视、文旅纪念与公共空间。
premium / restrained / cultural
core visual poster
核心意象
放生桥、课植园、漕港河、明清街巷、水墨层次与桥影留白；更强调“被凝视的江南”，突出历史感与文化厚度。
色彩
黛瓦墨青 石桥暖灰 宣纸米金 宣纸白
字体
Display / Body
Noto Serif SC · SemiBold + Noto Sans SC
应用侧重
Logo、文化海报、导视或文旅应用、明信片、公共家具。整体偏安静、偏慢、偏空间化和收藏感。

对比摘要

四项主差异一眼可见：年轻方案更像传播内容系统，高端方案更像文化空间系统。

core imagery

水网、打卡、周末游、焕新感，强调参与性和移动端传播效率。

color logic

青绿为主、银杏黄点睛，整体更明亮、更轻快，适合社媒传播和年轻人视角。

same place · two systems 同一朱家角，
两种品牌姿态。

左侧偏“会传播”，右侧偏“会沉淀”。

core imagery

桥、园、街巷、水墨留白，强调历史秩序、文脉密度与空间仪式感。

color logic

墨青与暖灰、米金构成克制底色，整体更安静，适合导视、纪念品与公共空间。

typeface

思源黑体 Heavy + Regular：字面干净、屏幕友好，适合短视频封面、海报与 APP。

application focus

Logo、主视觉海报、社媒传播图、宣传册、旅游 APP；更强调内容扩散与转化。

difference statement

不是同一套标志的换色版本，而是两套独立的视觉系统：一个向外发声，一个向内沉淀。

typeface

思源宋体 SemiBold + 思源黑体 Regular：更有书卷气和历史感，适合文化传播与空间识别。

application focus

Logo、文化海报、导视、明信片、公共家具；更强调场域感、纪念性与材料表达。
