# Review · 方案一：年轻化传播视觉方案 · v1

## 设计预期（从 brand-spec / deliverables 提取）

- 调性关键词：年轻、轻快、水感、可分享、现代文旅。
- 视觉气质：清透、灵动、轻量界面感、城市周末游、年轻打卡。
- 各子产物规格要求：Logo 需为 1024×1024 PNG 扁平矢量风；海报需为 1200×1600 HTML+PNG，体现“小桥、流水、人家”、古镇焕新和周末游吸引力；社媒图需为 1080×1080 HTML+PNG，突出“最江南”“水乡打卡”“年轻周末游”；宣传册需为 2480×1754 三折页展开版 HTML+PNG，包含核心景点、游船、北大街、课植园与推荐动线；APP 界面需为 390×844 HTML+PNG，包含景点卡片、路线、票务/导览入口。
- 核心预期：
  1. 整套方案应使用清水青绿 + 银杏黄方向，并保持轻盈、年轻、可分享的水乡气质。
  2. 各子产物要围绕朱家角的桥、水网、北大街、课植园等在地意象展开，而不是泛化旅游模板。
  3. 信息层级应优先服务“年轻周末游”的第一眼传播，再承接路线、景点和功能信息。
  4. APP、社媒、海报、宣传册之间应形成统一但不机械重复的年轻化系统语言。
