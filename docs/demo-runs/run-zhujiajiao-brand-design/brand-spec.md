## 色彩参考

- 方案一 Primary: 清水青绿（湖水青、竹青方向） [from-fact: 调研中高频意象为江南水、水乡、淀山湖、黛瓦、明清街、石桥、银杏、古镇焕新]
- 方案一 Accent: 活力银杏黄 / 旅游节亮黄 [from-fact: 调研中高频意象为江南水、水乡、淀山湖、黛瓦、明清街、石桥、银杏、古镇焕新]
- 方案二 Primary: 黛瓦墨青 / 深水墨色 [from-fact: 调研中高频意象为江南水、水乡、淀山湖、黛瓦、明清街、石桥、银杏、古镇焕新]
- 方案二 Accent: 石桥暖灰 / 宣纸米金 [from-fact: 三十六座石古桥如虹跨水]
- Background: 宣纸白、雾灰、浅米色 [inferred: 江南古镇与高端文化物料需要温润留白，年轻化传播也需保持轻盈可读]
- Ink: 深墨黑 / 乌篷灰 [inferred: 中文信息密度较高，需用高对比墨色保证海报、导视与 APP 可读性]

## 字体

- 方案一 Display: Noto Sans SC / 思源黑体 Heavy [inferred: 年轻化传播和 APP 界面需要现代、清晰、屏幕友好的无衬线标题]
- 方案一 Body: Noto Sans SC / 思源黑体 Regular [inferred: 社媒、宣传册与旅游 APP 需要高可读中文正文]
- 方案二 Display: Noto Serif SC / 思源宋体 SemiBold [inferred: 高端文化方向需体现历史、书卷与江南文脉气质]
- 方案二 Body: Noto Sans SC / 思源黑体 Regular [inferred: 导视与文旅应用需要稳定可读，避免全宋体造成小字号识别困难]

## 调性

- 方案一：年轻, 轻快, 水感, 可分享, 现代文旅
- 方案二：高端, 克制, 诗性, 历史厚度, 江南文脉
- 总体：文化, 历史, 古镇底蕴, 现代化表达

## 视觉气质

- 方案一：清透, 灵动, 轻量界面感, 城市周末游, 年轻打卡
- 方案二：留白, 细腻, 水墨层次, 园林秩序, 文博级质感

## 反 slop 禁区

- 紫渐变（除非 brief 明确要）
- emoji 图标
- 不得把两套方案做成同一套 logo 换色或同一系统下的两个传播方向
- 不得使用泛江南符号堆砌（随机灯笼、祥云、折扇、红金宫廷风）替代朱家角的放生桥、北大街、课植园、漕港河、明清街巷等核心意象
- 不得将色彩做成过饱和国潮红金或廉价旅游景区招牌风；高端文化方向尤其需要克制留白
