# 上海朱家角品牌形象设计 设计交付

## brief 摘要

- 本次任务面向上海朱家角古镇的文旅推广，要求围绕文化、历史、古镇底蕴进行现代化表达。
- 用户明确要求产出两套独立视觉方案，而不是同一系统下的换色分支：一套偏年轻化传播，一套偏高端文化品牌。
- 年轻化方案聚焦社媒传播、周末游打卡与 APP 友好体验；高端方案聚焦文化旅游、高品质导览与公共空间应用。
- 核心意象需来自调研事实，包括放生桥、北大街、课植园、漕港河、明清街巷与“小桥、流水、人家”的朱家角在地特征。
- 色彩方向不由用户预设，而由调研后提炼的水乡青绿、石桥灰、黛瓦墨青、宣纸米白等线索决定。

## 设计依据

- 方案一（年轻化传播）：以清水青绿为主色、银杏亮黄为强调色，配合宣纸白 / 雾灰背景与深墨黑信息色；使用 `Noto Sans SC / 思源黑体`，调性为年轻、轻快、水感、可分享、现代文旅。
- 方案二（高端文化）：以黛瓦墨青 / 深水墨色为主色、石桥暖灰 / 宣纸米金为强调色，配合宣纸白 / 浅米留白；标题侧使用 `Noto Serif SC / 思源宋体`，正文用 `Noto Sans SC` 保证可读，调性为高端、克制、诗性、历史厚度、江南文脉。
- 两套方案都以朱家角真实意象为基础，避免泛江南装饰堆砌、廉价旅游招牌感或同一 logo 换色复用。
