# Design Brief

## 主题
Foundry Lab

## 用户原话
> 为「Foundry Lab」设计品牌文案：一所专注于材料科学与制造工艺的工坊式研究所，定位是工程师/工匠/研究者三栖。气质：克制、动手做、不纸上谈兵。只做品牌文案任务（slogan + 简介 + 应用文案），不要做 logo/海报/UI，每个任务一轮，最快路径产出 + 汇总。

## 核心信息
- 是什么：材料科学与制造工艺的工坊式研究所，产学研结合的实战型机构
- 做什么：面向制造业的工艺研发、人才培训、技术转化，强调亲手实践而非理论空谈
- 给谁看：工程师（需要实战进阶）、工匠（需要理论升华）、研究者（需要落地场景）

## 调性关键词（5 个）
- 克制 / 务实 / 动手 / 深厚 / 可靠

## 必含元素
- slogan（短句，有力）
- 品牌简介（1-2 段话）
- 应用文案（官网标题 / 社交媒体简介 / 宣传单页一句）

## 禁含元素
- 禁浮夸修辞（不用"引领"、"颠覆"、"创新"等空洞词）
- 禁过度设计感（不做 logo/海报/UI）
- 禁学术八股（不堆砌术语，要人话）
