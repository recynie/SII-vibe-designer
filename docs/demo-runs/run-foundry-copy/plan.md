# Plan - Foundry Lab 品牌文案

## 任务概览
只做品牌文案任务（slogan + 简介 + 应用文案），每个任务一轮，最快路径产出。

## 子任务清单

| # | 任务名 | Skill | 输出文件 | 备注 |
|---|---|---|---|---|
| 1 | Slogan | copywriting | artifacts/copy/v1.txt | 短句，有力，克制务实 |
| 2 | 品牌简介 | copywriting | artifacts/copy/v1.md | 1-2 段话，工程师/工匠/研究者三栖 |
| 3 | 应用文案 | copywriting | artifacts/copy/v1.md | 官网标题 + 社交简介 + 宣传句 |

## 执行顺序
按顺序串行执行，每完成一个任务立即调用 critic 评审，通过则继续下一个。

## 验收标准
- 反 AI slop：禁空洞口号、禁浮夸修辞、禁学术八股
- 气质匹配：克制、动手做、不纸上谈兵
- 交付格式：纯文本文件，清晰列出每条文案
