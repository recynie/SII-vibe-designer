# Foundry Lab 品牌文案设计交付

## Brief 摘要
为 Foundry Lab——一所专注于材料科学与制造工艺的工坊式研究所——设计品牌文案。定位工程师/工匠/研究者三栖，气质要求克制、动手做、不纸上谈兵。只做 slogan + 简介 + 应用文案三项，每个任务一轮通过，最快路径产出。

## 设计依据
引用 brand-spec.md 关键决策：
- 色板方向：炭黑 #1A1A1A + 铜橙 #C45C26 accent，体现克制与工匠温度
- 字体方向：思源衬线（标题）+ 思源黑体（正文），稳重传承
- 视觉气质参考：德国工业设计派 + 日本职人文化，克制务实、功能本位
- 反 AI slop：禁空洞口号（颠覆/创新/引领）、禁浮夸修辞、禁学术八股
