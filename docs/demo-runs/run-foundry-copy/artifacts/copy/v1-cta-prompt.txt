# 应用文案创作思路

## 创作依据

来自 brief.md 核心信息：
- Foundry Lab 定位：材料科学与制造工艺的工坊式研究所
- 目标人群：工程师 / 工匠 / 研究者三栖
- 气质要求：克制、动手做、不纸上谈兵
- 调性关键词：克制 / 务实 / 动手 / 深厚 / 可靠
- 禁含元素：禁浮夸修辞（"引领"、"颠覆"、"创新"）、禁学术八股

来自 brand-spec.md 视觉气质：
- 克制留白、硬朗线条、材料质感、低饱和度、手工温度

---

## 逐条说明

### 1. 官网标题：把材料，变成答案

**出发点**：一句话说明 Foundry Lab 是做什么的
- "把材料"扣住材料科学主业
- "变成答案"暗示从理论到实践的转化
- 句式简短有力，动词"变成"有动手感
- 克制：没有"引领"、"创新"等空洞词
- 务实：直接点出核心动作（把X变成Y）

### 2. 社交媒体简介

**出发点**：30-50字说明身份和差异化
- 第一句定义身份："材料科学与制造工艺的工坊式研究所"
- 第二句说明给谁、做什么："工程师、工匠、研究者，在这里动手做实事"
- 结尾点题："不纸上谈兵"呼应brief中反复强调的"动手做"vs"空谈"
- 总计43字，符合30-50字范围

### 3. 宣传单页一句：别坐而论道，来工坊动手

**出发点**：有号召力但不虚
- "别坐而论道"是反着说，点出行业痛点（太多空谈）
- "来工坊"呼应"工坊式研究所"的定位
- "动手"直接呼应"动手做"的核心气质
- 对仗工整，语气有力但不夸大
- 没有用"颠覆"、"创新"等词

---

## 风格统一性

三则文案共同遵循：
- 动词主导（变成、动手、来），体现行动感
- 短句优先，不绕弯子
- 对"空谈"的批判一以贯之
- 无空洞术语，无过度修辞
