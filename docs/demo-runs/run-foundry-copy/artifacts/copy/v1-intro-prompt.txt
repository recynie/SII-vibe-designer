# 设计依据

## 策略思考

1. **受众锚定**：工程师（需要实战进阶）、工匠（需要理论升华）、研究者（需要落地场景）——三类人共同特征是务实、看重可验证的结果。文字必须给出具体价值，而非抽象承诺。

2. **气质契合**：克制、动手做、不纸上谈兵。
   - 不写"引领"、"颠覆"等空洞词
   - 不堆砌学术术语
   - 强调"从真实生产场景中来，到可落地技术中去"——用行动而非修辞定义自己

3. **差异化定位**：
   - 传统科研机构：论文导向、理论先行
   - Foundry Lab：问题导向、实践验证
   - 差异点落在"工坊"和"连接三方"

## 写法选择

- 第一段：直接回答"是什么、做什么"，不绕弯子
- 第二段：面向三类受众分别给出具体价值点，证明"三栖"定位不是概念

## 禁含元素自检

- ❌ 学术八股：无堆砌术语，用"材料科学与制造工艺"而非"先进材料制备与智能制造工艺"
- ❌ 空洞修辞：无"引领"、"颠覆"、"创新驱动"
- ❌ 高冷姿态：结尾落在"让技术真正产生价值"，是务实者的语言
