# Slogan 设计依据

## 整体思路

 Foundry Lab 的核心定位是「材料科学与制造工艺的工坊式研究所」，气质关键词为「克制、动手做、不纸上谈兵」。基于这一认知，Slogan 设计遵循以下原则：

- **具象胜于抽象**：不用「创新」「颠覆」「引领」等空洞词，而是回到「手」「材料」「事」等具体意象
- **行动导向**：强调「做」而非「想」，契合工坊属性和三栖定位
- **克制有力**：短句收尾，不加感叹号，不堆砌修辞

## 变体解析

**A · 真知，在手上。**

 定位：务实动手。以「手」为核心意象，直接呼应「动手做」气质。「真知」点明研究所的学术属性，但用「手上」而非「书中」完成对比——不说空话，只写真东西。

**B · 用材料，回答问题。**

 定位：工艺传承。「材料」是 Foundry Lab 的核心资产，「回答问题」暗示这里解决的是真实的工程问题，而非纸上谈兵。整体有日本职人文化的影子——用手艺说话。

**C · 不说空话，只做事。**

 定位：工匠本色。最直白的表达，但正因为直白反而有力。明确拒绝了「学术八股」和「浮夸修辞」，是一种自我宣示。

**D · 三十年薪火，手上传授。**

 定位：深厚积淀。引入时间维度（「三十年」），暗示积累与传承。「手上传授」呼应工坊的师徒制感，也与品牌色的「工匠温度」呼应。

**E · 做得到，才开口。**

 定位：可靠承诺。这是一条略带性格的 Slogan——不吹牛，只说自己能做到的。与「克制」「可靠」两个关键词高度吻合。

## 禁含元素自检

- ❌ 「创新」「颠覆」「引领」：未使用
- ❌ 空洞修辞：全部使用具体动词（做、回答、传授、开口）
- ❌ 感叹号收尾：全部以句号或静默语气收束
- ❌ 英文直译：全部为原生中文表达

## 与 Brand Spec 的呼应

- 色板中的「铜橙色 #C45C26」代表工匠温度——Slogan 中的「手」「材料」「事」等意象呼应这一温度
- 字体方向「衬线体做标题」——Slogan 作为标题级文字，适合使用 Noto Serif SC 等衬线体呈现
- 视觉气质「硬朗线条」——文字层面的硬朗体现为：少修饰、短句、主动词
