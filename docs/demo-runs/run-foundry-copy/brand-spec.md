# Brand Spec

> 采集日期：2026-05-16
> 来源：推断（基于用户 brief 描述）

## 色板
- Primary: #1A1A1A  <理由：炭黑，克制、沉稳、工坊感>
- Secondary: #4A4A4A  <理由：深灰，工业材料气质>
- Background: #F5F5F0  <理由：米白/纸白，还原纸张/工作台质感>
- Ink (正文): #2D2D2D  <理由：近黑正文，清晰但不高冷>
- Accent: #C45C26  <理由：铜橙色，工匠温度、铸造感，克制使用>

（不超过 5 个，禁用紫渐变除非 brief 明确要求）

## 字体方向
- Display: 衬线体，推荐「Noto Serif SC」或「Source Han Serif CN」—— 稳重、有工艺传承感，适合标题与 slogan
- Body: 无衬线，推荐「思源黑体 / Noto Sans SC」—— 清晰、中性、屏幕阅读友好

## 视觉气质
3-5 个形容词，与"调性关键词"一致但描述视觉表现：
- 克制留白：不多做装饰
- 硬朗线条：拒绝圆滑、圆润
- 材料质感：呈现金属/木材/纸张的本来面目
- 低饱和度：除 accent 外整体偏暗
- 手工温度：不是冰冷的高科技，是有烟火气的制造业

## 设计参考方向
- 方向：德国工业设计派（Dieter Rams 体系）+ 日本职人文化（柳宗理 Sori Yanagi）
- 理由：克制、务实、功能本位、不做多余装饰；与"工程师/工匠/研究者三栖"身份吻合——既是理性的工程师，也是感性的工匠

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变、emoji 图标、generic stock 图库、亮蓝色科技感
- 不要：口号式"颠覆创新"、空洞的"引领未来"
- 不要：过度抛光的无菌感（工坊应该是带痕迹的）
