# Brand Spec

> 采集日期：2026-05-16
> 来源：WebSearch - 维基百科/朱家角镇词条

## 色板
- Primary: #1E3A5F （理由：深墨蓝，与水墨气质呼应但比传统墨色更现代）
- Secondary: #6B9AC4 （理由：淡青水色，呼应江南水乡的河网意象）
- Background: #FAF8F5 （理由：米白纸色，模拟宣纸/绢帛质感）
- Ink (正文): #2D3436 （理由：炭灰，比纯黑柔和，契合水墨画的淡墨层次）
- Accent: #C47F5A （理由：淡褐，提取自古镇石桥/老宅的岁月痕迹色）

## 字体方向
- Display: 思源宋体 / 方正正宋 / 汉仪风雅宋 — 具备宋体骨架但笔画干净的当代中文衬线体，1-2款备选
- Body: 思源黑体 / 苹方 / Noto Sans CJK SC — 无衬线系统字体，避免授权风险

## 视觉气质
- 水墨留白：大量负空间，忌满版填充
- 轻盈渗透：纹理以半透明叠加呈现，不做死板描边
- 笔触感：辅助图形带书法/写意笔触，但克制面积
- 静谧诗意：低饱和度配色，避免喧闹的商业感

## 设计参考方向
- 韩湛宁（或 HK+）— 东方极简+当代平面设计语言，适配"水墨不刻板"的诉求
- 田中良治（日本）— 留白与信息节奏的克制美学
- 选一方向并说明：韩湛宁/本土当代派更适合本项目，因朱家角需要"中国籍的当代感"而非纯粹的日式极简

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变、emoji 图标、generic stock 古镇照片、Inter/ Roboto 当 display
- 不要：大红大金的高饱和度节庆配色
- 不要：过度堆砌的传统纹样（祥云/龙凤/回纹等通通慎用）
- 不要：做成"江南古镇通用模板"的样貌，需体现朱家角的独特辨识度
