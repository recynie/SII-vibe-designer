# Design Brief

## 主题
朱家角古镇文旅品牌形象设计

## 用户原话
> 请为上海朱家角古镇做一套文旅品牌形象设计。朱家角是江南水乡六大古镇之一，明清建筑、河道纵横、桥梁林立。这次设计目标是面向年轻一代游客（25-35）做一次品牌焕新，既要保留古镇的水墨气质，又不能太刻板老气。

## 核心信息
- 是什么：朱家角是上海青浦区的江南水乡古镇，明清古建筑群保存完整，河道纵横、桥梁林立的典型水乡风貌
- 做什么：面向25-35岁年轻游客的文旅品牌焕新，定位为"可抵达的江南诗意生活"
- 给谁看：25-35岁城市青年，追求文化体验、审美消费、逃离都市喧嚣的周末短途游用户

## 调性关键词（5 个）
- 水墨 / 诗意 / 轻盈 / 摩登 / 静谧

## 必含元素
- 古镇标志性建筑意象（放生桥/北大街/河岸民居）
- 水乡纹理（水波/雨丝/石桥）
- 用户未指定，由设计师判断

## 禁含元素
- 刻板的传统古镇宣传照套路（大红灯笼+导游旗）
- 过度商业化的旅游纪念品感
- 浓重的民俗堆砌感

## 交付清单
最少包含：
- [ ] Logo 主标志
- [ ] 主视觉海报或应用物料一件
- [ ] 品牌文案（slogan + 一段简介）
- [ ] 文创产品命名 + 文案
- [ ] 旅游导览 H5 移动端首屏
