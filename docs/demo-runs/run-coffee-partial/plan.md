# Plan - 钝角咖啡品牌形象设计

## 任务清单

| # | 子任务 | 技能 | 预期输出 |
|---|--------|------|----------|
| 1 | Logo 主标志 | logo | artifacts/logo/v1.png + v1.prompt.txt |
| 2 | 品牌文案 | copywriting | artifacts/copy/v1.txt |
| 3 | 主视觉海报 | poster | artifacts/poster/v1.html + v1.png |
| 4 | 品牌官网首页 mockup | ui-mockup | artifacts/ui-mockup/v1.html |

## 执行顺序

**严格串行**：按顺序执行，一次只调度一个 designer，完成评审后再进入下一个。

1. 先做 Logo（品牌根基）
2. 再做品牌文案（支撑所有视觉物料）
3. 然后主视觉海报（应用文案）
4. 最后官网 mockup（综合展示）

## 评审规则

- 每轮最多 2 次迭代
- critic 评分标准：5 维度（一致性、视觉质量、信息传达、创意性、执行精度）
- 通过标准：5 维度平均 >= 7 分
- 不通过则进入第 2 轮，仍不通过则向用户提问
