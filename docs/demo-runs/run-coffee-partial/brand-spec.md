# Brand Spec

> 采集日期：2026-05-16
> 来源：推断（基于 brief 描述 + 行业调研）

## 色板
- Primary: #1A1A1A（深炭灰 - 呼应工业风基底，克制有力）
- Secondary: #8B5A2B（焦糖棕 - 咖啡温暖感，控制在低饱和度）
- Background: #F5F5F5（冷白 - 极简空间感）
- Ink (正文): #333333（深灰 - 清晰但不刺眼）
- Accent: #E8E8E8（浅水泥灰 - 工业材质感）

## 字体方向
- Display: 建议使用 Noto Serif SC（思源宋体）或 Playfair Display，搭配几何感中文字体如思源黑体 Heavy，强调「考究」与「克制」的平衡
- Body: 推荐使用 system-ui / -apple-system / Noto Sans SC（思源黑体），确保跨平台一致性

## 视觉气质
- 高对比：深炭灰与冷白形成有力视觉层次
- 留白克制：大量留白，不过度填充信息
- 几何精简：图形元素以几何形态为主，呼应工业风
- 冷幽默点缀：通过文案或图形细节传递「钝角」的幽默感

## 设计参考方向
- **推荐方向：日本极简主义（Kenya Hara 原研哉式）**
  - 理由：与品牌「克制、考究」的调性高度吻合，注重负空间运用，视觉冷静但有温度
- **备选方向：北欧功能主义**
  - 理由：强调实用性与材质感，适合工业风定位

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变、emoji 图标、generic stock 图库照片、Inter 当 display 字体
- 不要：过度圆润的边角设计（与工业风冲突）
- 不要：使用「可爱」「萌」风格的插画元素
- 不要：大量使用暖橙色系（焦糖棕仅作点缀）
