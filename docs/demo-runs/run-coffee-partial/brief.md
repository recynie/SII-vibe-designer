# Design Brief

## 主题
钝角咖啡（Obtuse Coffee）

## 用户原话
> 请为一家叫做「钝角咖啡」（Obtuse Coffee）的精品咖啡创业品牌做整套品牌形象设计。定位：第三波精品咖啡，自烘焙豆 + 单品手冲为主，店铺空间偏极简工业风，目标客群是 25-40 岁城市白领与设计/科技从业者。气质关键词：克制、考究、不浮夸、有点冷幽默。交付物至少包含：品牌 logo、主视觉海报、品牌文案（slogan + 简介 + 一段店内文案）、品牌官网首页 mockup（落地页）。

## 核心信息
- 是什么：第三波精品咖啡品牌，主打自烘焙单品豆与手冲体验，定位高品质专业咖啡
- 做什么：提供自烘焙精品咖啡豆、单品手冲饮品，店铺空间呈现极简工业风格
- 给谁看：25-40 岁城市白领，设计/科技从业者，审美素养高，偏好低调有内涵的品牌

## 调性关键词（5 个）
- 克制
- 考究
- 不浮夸
- 冷幽默
- 专注

## 必含元素
- 品牌 logo（含中英文名称）
- 主视觉海报
- 品牌文案：slogan + 品牌简介 + 店内文案
- 品牌官网首页 mockup（落地页形式）

## 禁含元素
- 过度装饰的视觉元素
- 暖色调为主的温馨风格（与品牌调性不符）
- 流行/快消感的设计语言
