## 改动说明

基于 v1.review.md 的反馈进行以下修改：

1. **颜色严格化**
   - v1 问题：Primary #2C3E50 偏差偏绿，Accent #4A7C59 未体现
   - v2 改进：prompt 中使用 "STRICTLY use only #2C3E50 (dark cyan-grey, Primary) - NO color shifts, NO warm/cool tints" 强制约束
   - v2 改进：强调 "STRICTLY use only #4A7C59 (sage moss green Accent)" 用于水波纹点缀

2. **增加"鲜活"感**
   - v1 问题：整体偏沉闷，未体现"鲜活"关键词
   - v2 改进：加入 "dynamic curved flowing lines"、"flowing water ripple"、"lightweight lines suggesting vitality and freshness" 等动态描述
   - v2 改进：使用 "dynamic yet elegant" 替代原有的静态描述

3. **苔绿点缀**
   - v1 问题：wordmark 和 combo 未使用 Accent #4A7C59
   - v2 改进：wordmark 中 "subtle dynamic water ripple lines flowing beneath the text in #4A7C59 (sage moss green Accent)"
   - v2 改进：combo 中 "flowing water ripple accent in #4A7C59 (sage moss green Accent) woven through the composition"

## 设计依据

- 使用 brand-spec.md Primary #2C3E50（黛青色）表达"诗意/精致"的古镇气质
- 使用 Accent #4A7C59（苔绿）作为江南园林植被的清新点缀和水流动态
- 背景使用 #F5F5F0（米白）呼应宣纸质感，承载水墨气质
- 形态选择：放生桥抽象化动态曲线 + 流动水波纹，融合古镇标志性元素与年轻化鲜活设计
- 参考方向：原研哉极简主义与东方美学融合风格，避免老气横秋的传统表达
