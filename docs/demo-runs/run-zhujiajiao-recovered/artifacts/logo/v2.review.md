## 改进建议

1. **[prompt 问题]** v2-wordmark 文字颜色完全错误 - 需在 prompt 中明确 "dark cyan-grey #2C3E50 for Chinese characters, STRICTLY NO light tints, NO pale colors, the text must be DEEP dark, not light grey or light green"
   - 当前问题：文字可能生成了淡绿/浅灰色，完全背离品牌色

2. **[prompt 问题]** 颜色偏差的根本原因是 AI 对 hex 理解不稳定 - 建议在 prompt 中加入视觉参考描述而非仅 hex 值：
   - Primary #2C3E50 = "deep charcoal with blue undertone, like wet slate"
   - Accent #4A7C59 = "muted sage green, like moss on garden stones"

3. **[素材问题]** v2-mark 和 v2-combo 中的水波纹颜色偏浅 - 实际 #56726a/#5c7270 偏灰偏亮，应为更深沉的苔绿。可考虑用更明确的 "dark sage green, NOT light, NOT greyish" 约束
