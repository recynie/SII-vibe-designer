# 执行计划：朱家角古镇文旅品牌形象设计

## 项目概述
- **目标**：面向25-35岁年轻游客的品牌焕新
- **调性**：诗意 / 精致 / 国际化 / 清新 / 鲜活
- **核心挑战**：保留水墨江南气质的同时避免刻板老气

## 子任务清单

### 1. 文旅 Logo 设计
- **skill**: logo
- **输出**: artifacts/logo/v1.png + v1.prompt.txt
- **要求**: 体现"水墨江南"意境 + 年轻化现代感，融合古镇标志性元素（放生桥/小桥流水）

### 2. 主视觉海报
- **skill**: poster
- **输出**: artifacts/poster/v1.html + v1.png
- **要求**: 展现古镇水墨气质，适用于社交媒体传播（16:9比例）

### 3. 文创产品命名 + 文案
- **skill**: copywriting
- **输出**: artifacts/copy/v1.md
- **要求**: 3-5款文创产品命名 + 卖点文案，风格诗意且年轻化

### 4. 旅游导览 H5 首屏
- **skill**: ui-mockup
- **输出**: artifacts/ui-mockup/v1.html
- **要求**: 移动端首屏设计，包含古镇氛围图、slogan、导航入口

## 执行顺序
1. Logo → 2. 海报 → 3. 文案 → 4. H5
- 每个任务由 designer 执行 → critic 评审（最多2轮）
- 串行执行，确保质量
