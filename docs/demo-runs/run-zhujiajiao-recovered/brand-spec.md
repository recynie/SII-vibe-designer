# Brand Spec

> 采集日期：2026-05-16
> 来源：WebSearch（百度百科、青浦区政府官网、澎湃新闻）

## 色板
- Primary: #2C3E50（黛青色，呼应水墨江南的白墙黛瓦基调）
- Secondary: #8B9DC3（浅灰蓝，江南天空与河水的柔和意象）
- Background: #F5F5F0（米白，宣纸质感，承载水墨气质）
- Ink: #1A1A1A（墨黑，正文主色）
- Accent: #4A7C59（苔绿，江南园林植被的清新点缀）

## 字体方向
- Display: 
  - 思源宋体 CN Heavy（正式感与水墨气质兼具）
  - 方正筑紫古典宋（古典韵味，可做标题装饰）
  - 推荐备用：站酷高端黑（现代年轻感）
- Body: 
  - 思源黑体 CN Regular（清晰现代，无授权风险）
  - 系统默认：-apple-system, BlinkMacSystemFont, "PingFang SC"

## 视觉气质
- 留白克制：大面积留白，呈现水墨画的呼吸感
- 线条轻盈：避免粗重线条，用细线勾勒江南建筑轮廓
- 色彩低饱和：以灰调为主，accent 色彩作为点睛
- 动静结合：静态呈现水墨意境，细节处可融入年轻化的动态元素

## 设计参考方向
- 原研哉（Kenya Hara）：极简主义与东方美学的融合，擅长在简洁中传递深层意象
- 为什么会适配本 brief：朱家角的核心是"水墨江南"，原研哉的设计哲学强调"空"与"白"，与古镇的意境高度契合，同时其作品具有国际化的现代感，能满足年轻游客的审美偏好

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变（江南古镇色彩体系中不存在紫色系）
- 不要：emoji 图标（破坏水墨意境的整体感）
- 不要：generic stock 照片（需原创或精心挑选的古镇素材）
- 不要：过度使用传统纹样堆砌（显得老气横秋）
- 不要：Inter / Roboto 等西文字体作为 Display（中文品牌需要有匹配的中文字体支撑）
