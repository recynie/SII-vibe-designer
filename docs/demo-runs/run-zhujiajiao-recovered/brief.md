# Design Brief

## 主题
上海朱家角古镇文旅品牌形象焕新

## 用户原话
> 请为上海朱家角古镇做一套文旅品牌形象设计。朱家角是江南水乡六大古镇之一，明清建筑、河道纵横、桥梁林立。这次设计目标是面向年轻一代游客（25-35）做一次品牌焕新，既要保留古镇的水墨气质，又不能太刻板老气。交付物至少包含：文旅 logo、宣传海报主视觉、文创产品命名 + 文案、旅游导览 H5 移动端首屏。

## 核心信息
- 是什么：拥有1700多年历史的江南水乡古镇，上海四大历史文化名镇之一，2024年接待游客超547万人次
- 做什么：面向25-35岁年轻游客的文旅品牌焕新，定位"中国古镇上海入境旅游第一站"，打造"最江南"文化品牌
- 给谁看：25-35岁年轻一代游客，偏好深度体验、追求品质感与文化氛围的城市白领及小资群体

## 调性关键词（5 个）
- 诗意 / 精致 / 国际化 / 清新 / 鲜活

## 必含元素
- 水墨江南意象（白墙黛瓦、小桥流水）
- 古镇标志性建筑元素（放生桥为江南最大五孔石桥）
- 年轻化视觉语言（避免老气横秋）

## 禁含元素
- 紫渐变 / emoji 图标 / generic stock 图库素材
- 过度传统老旧的表现手法（如直接堆砌传统纹样）
- 刻意做旧的粗糙质感

- [ ] 主视觉海报
- [ ] 文创产品命名 + 文案
- [ ] 旅游导览 H5 移动端首屏
