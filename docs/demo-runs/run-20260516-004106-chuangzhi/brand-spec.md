# Brand Spec

> 采集日期：2026-05-16
> 来源：用户 brief 推断 + 竞品调研（商汤研究院、复旦 AI 研究院等公开信息）

## 色板
- Primary: #1A2B4A（墨蓝色，学术稳重，奠定信任感）
- Secondary: #2D3E5F（深灰蓝，辅助沉稳）
- Background: #F7F9FC（冷白，近乎白色的极浅灰，营造现代洁净感）
- Ink (正文): #1A1A1A（近黑色，正文首选，对比清晰）
- Accent: #00B8A9（青绿，点缀色，象征 AI 活力与未来感）

## 字体方向
- Display: **思源黑体 Heavy** 或 **Noto Sans SC Black**（无衬线、现代、权威感）
- Body: **思源黑体 Regular** / **Noto Sans SC**（系统安全、中性可靠）
- 备选：**Inter**（如需要更偏西方现代感时使用，但避免单独用作 Display）

## 视觉气质
3-5 个形容词（与调性关键词呼应）：
- 深沉内敛（墨蓝色主调）
- 留白克制（大量呼吸空间）
- 几何精准（少装饰、重结构）
- 冷青点缀（青绿作为唯一的"活跃"元素）
- 理性秩序（网格系统清晰、信息层级分明）

## 设计参考方向
- **推荐方向：Pentagram 信息建筑派**
  - 理由：Pentagram 擅长将复杂信息结构化，适合"研究院+高等教育"混合体这一需要同时传递学术深度与教育亲和力的场景。创智学院不是纯商业品牌，需要冷静、克制、有体系感的视觉语言。

- **备选方向：Kenya Hara 东方极简**
  - 理由：如需更强调"东方智慧"与"禅意科研"气质时可用，但需谨慎——创智学院定位更偏技术前沿而非传统哲学。

## 反 AI slop 禁区（针对本项目）
- 不要：紫渐变（行业通用 Slop）
- 不要：emoji 图标
- 不要：generic 科技感 stock 插图（如握手/灯泡/火箭）
- 不要：圆角过度（保持锐利、克制）
- 不要：使用 Inter 作为唯一 Display 字体（需搭配更有分量的中文衬线或重型无衬线）
