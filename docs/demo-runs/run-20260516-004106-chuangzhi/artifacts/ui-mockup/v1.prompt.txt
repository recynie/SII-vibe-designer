# 创智学院官网首页 Mockup - 设计依据

## 色彩系统（来自 brand-spec.md）
- Primary: #1A2B4A（墨蓝色，学术稳重）
- Secondary: #2D3E5F（深灰蓝，辅助）
- Background: #F7F9FC（冷白，极浅灰）
- Ink: #1A1A1A（近黑色，正文）
- Accent: #00B8A9（青绿，点缀色）

## 字体配对
- Display: Noto Sans SC Black / Heavy（标题）
- Body: Noto Sans SC Regular（正文）

## 风格方向
- **Pentagram 信息建筑派**：网格系统清晰，信息层级分明
- 极简克制：大量留白，无多余装饰
- 几何精准：卡片、按钮均为直角/小圆角
- 理性秩序：使用精确的 padding/margin 数值

## 页面结构
1. **Header**：Logo + 导航（研究/人才培养/关于我们/联系我们）+ CTA
2. **Hero**：标题 + 一句话定位 + 双 CTA 按钮 + 几何图形视觉
3. **机构简介**：Section label + 标题 + 介绍文字 + 统计数据
4. **核心方向**：4 个研究领域卡片（机器学习/CV/NLP/AI系统）
5. **最新动态**：3 条新闻卡片
6. **Footer**：品牌信息 + 链接分组 + 社交图标

## 反 AI Slop 自检
- ✅ 无紫渐变
- ✅ 无 emoji 图标
- ✅ 无 generic stock 插图（使用几何图形代替）
- ✅ 无过度圆角（使用小圆角或直角）
- ✅ 无"赋能"类营销文案
- ✅ 无左侧彩色 border accent 卡片

## 技术实现
- 纯 HTML + CSS，无 JavaScript 依赖
- CSS Grid + Flexbox 布局
- 响应式设计基础（固定 1440px 宽度）
- 使用 CSS 变量管理色彩系统
