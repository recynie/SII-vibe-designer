## 改进建议（建议项，非阻塞）

1. **[版式问题]** 如需真·响应式，建议添加基础媒体查询：
   ```css
   @media (max-width: 1024px) {
     html, body { width: 100%; }
     .hero { grid-template-columns: 1fr; padding: 80px 40px; }
     .research-grid, .news-grid { grid-template-columns: repeat(2, 1fr); }
   }
   ```

2. **[素材问题]** Hero 区域视觉目前是纯CSS几何图形（方形+边框），如需更丰富的视觉，可考虑加一张抽象的 AI/数据可视化类配图（但要避免 generic stock 风格）
