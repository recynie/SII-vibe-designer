# 主视觉海报设计依据

## 设计决策

### 1. 版式选择
- **横版 1200×628px**：适合微信/微博/官网 banner 投放
- **左右分栏结构**：左侧文字信息，右侧视觉区，符合 Pentagram 信息建筑派理念

### 2. 色彩运用
- Primary #1A2B4A：主标题 + 右侧背景，奠定学术稳重基调
- Accent #00B8A9：Logo 图形 + 几何线条 + 强调线，点缀未来感
- Background #F7F9FC：整体底色，营造洁净现代感
- Secondary #2D3E5F：副标题和描述文字，辅助沉稳阅读

### 3. 字体配对
- 标题：Noto Sans SC Heavy（900），72px，强调力量感与权威
- 正文：Noto Sans SC Regular（400），15px，清晰易读

### 4. 几何元素
- 右侧使用抽象几何装饰（方形、线条），呼应"信息建筑派"风格
- 几何图形以 #00B8A9 勾边，活跃版面但不喧宾夺主

### 5. 配图
- 抽象几何建筑可视化，深蓝 + 青绿调
- 生成工具：gen_image.py (minimax backend)
- 与整体色调协调，避免紫渐变等 AI slop

## 反 Slop 自检
- ✅ 无紫渐变
- ✅ 无 emoji
- ✅ 无 generic stock 插图
- ✅ 几何精准、留白克制
- ✅ 思源黑体 Heavy + Regular 搭配
