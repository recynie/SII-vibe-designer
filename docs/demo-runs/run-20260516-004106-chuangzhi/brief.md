# Design Brief

## 主题
创智学院

## 用户原话
> 创智学院是上海一所专注于人工智能创新与人才培养的研究型机构，定位偏向研究院 + 高等教育混合体，面向青年研究者和工程师，希望气质上既有学术稳重感、又有面向未来的现代感。

## 核心信息
- 是什么：上海一家专注于人工智能创新与人才培养的研究型机构，兼具研究院与高等教育双重属性
- 做什么：AI前沿研究、青年研究者和工程师培养、技术成果转化
- 给谁看：青年研究者（硕士/博士研究生）、年轻工程师、潜在合作伙伴（企业/高校）

## 调性关键词（5 个）
- 学术 / 稳重 / 前沿 / 理性 / 克制

## 必含元素
- Logo 主标志（需体现"创智"概念，可融入 AI/科技隐喻）
- 品牌色系（深蓝+冷灰+青绿点缀）
- Slogan：一句话表达机构定位与愿景
- 品牌简介：100-150 字机构介绍

## 禁含元素
- 避免过度娱乐化、萌系元素
- 禁用紫渐变（AI 行业通用 Slop）
- 禁用 emoji 图标
- 禁用 generic stock 风格插图
