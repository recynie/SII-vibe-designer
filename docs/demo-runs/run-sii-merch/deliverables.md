  - 徽章 | 圆形/异形徽章 mockup PNG + 徽章平面 PNG，专业招生纪念感，使用简洁品牌蓝符号系统。
  - 笔记本 | 笔记本封面 mockup PNG + 封面平面 PNG，科技蓝封面、结构化标题与抽象前沿科研图形。

  - 水杯 | 水杯 mockup PNG + 杯身展开平面 PNG，明亮日用风格，图形与科技招生系列明显区分。

## 隐式
- 系列口号与辅助文案 | 纯 markdown 文案稿，为两个系列各提供 3 条可上产品的短口号与辅助文字；必须只使用已确认的“上海创智学院”“SII”“Shanghai Innovation Institute”，不得编造校训、年份、院系。推断理由：brief 允许系统自行提炼口号与辅助文字，且 12 个子产物需要统一文案母题。
- 文创印刷应用规范页 | HTML 排版 + 渲染 PNG，汇总两系列色板、字体、logo 使用、图形母题、禁用信息和 6 类载体应用规则；推断理由：brief 要求每类产品兼顾 mockup 与可印刷/延展平面稿，规范页可保证 12 个子产物延展一致性。

## 拒绝
- 传统校徽/院徽重设计 | brief 明确避免传统高校校徽式设计，且本 run 聚焦文创产品而非重塑官方标志。
- 视频广告或动效物料 | brief 要求照片风格产品 mockup 与平面设计稿，视频/动效超出平面设计交付范围。
- 真实打样生产文件 | 本 run 只输出视觉 mockup、平面设计稿与规范页，不负责印厂刀模、材质采购或生产落地。

## 决策依据
- 显式条目来自 brief「产品范围：共 6 类——帆布袋、T 恤、贴纸、徽章、笔记本、水杯」与 facts.md「系列结构明确为 2 个风格系列，每个系列覆盖 6 类产品，共 12 个文创设计方向/子产物」。
- 两个显式套装采用 sub-items，是因为每个系列共享同一视觉基础，并分别覆盖 6 个独立产品文件组，符合“同一风格的多载体延展”归组模式。
- mockup 与平面稿规格来自 facts.md「每类产品希望兼顾真实场景、照片风格的产品 mockup 效果图，以及可用于印刷/延展的平面设计稿」。
- 科技招生系列依据 facts.md「科技风格偏招生宣传、品牌专业感」；校园青春系列依据 facts.md「校园青春风格偏新生纪念、日常使用感」。
- 使用 assets/sii-logo.svg 的依据是 facts.md「上海创智学院官网使用 logo.svg 作为页眉标志」。
- 隐式口号文案依据 facts.md「文案可由系统自行提炼；若出现专有名词，必须保证“上海创智学院”、“SII”等名称准确」。
- 隐式规范页依据 brief 对系列统一、系列差异、印刷/延展稿的要求；开放式文创套装需要规范页帮助 designer 与 critic 评估一致性。
- 拒绝条目依据 facts.md「明确排除幼稚、低龄化设计，避免传统高校校徽式设计，禁止使用不确定的校训、年份、院系等信息」以及本 run 平面设计范围。
