PRINT APPLICATION GUIDE · MERCHANDISE SYSTEM

文创印刷应用规范页

用于上海创智学院两套文创系列的印刷延展：科技招生系列保持冷静、专业与结构化；校园青春系列保持清爽、日用与新生友好。所有载体只使用已确认的「上海创智学院」「SII」「Shanghai Innovation Institute」。

Logo使用官网标志资产，不重绘、不改变比例、不拆分图文关系。

01 · 两系列色板

科技招生系列

主色为品牌蓝与官网主蓝，适合深色或冷白底；图形语言使用网格、数据流、芯片线路与空间透视，强调招生宣传与专业可信。

校园青春系列

以浅蓝灰底、品牌蓝和活力橙形成差异；采用手账式模块与创客日常符号，保持年轻但避免低龄卡通。

02 · 总则

A

印刷面积纪律

单个载体主图形占可印刷区域 35%–65%；贴纸和徽章可提高至 75%，但保留 10% 以上安全边。

B

系列区分

科技招生系列不使用橙色作大面积背景；校园青春系列不使用黑蓝重底和高密度线路满版。

C

文字信息

中文名称必须写作「上海创智学院」；英文只使用 SII 或 Shanghai Innovation Institute。

03 · 字体与 Logo 使用

Typography

montserrat-medium

sourcehansanscn-regular 用于中文正文、载体说明与辅助信息。标题与正文层级保持 2.5× 以上落差。

Logo lockup

四周留白 ≥ 标志图形高度的 0.25×；最小印刷宽度建议 ≥ 28mm。禁止拉伸、描边、重排或与新图形合成校徽。

05 · 6 类载体应用规则

帆布袋

正面竖向构图；主图形居中或上移 8%。科技版用蓝色网格主视觉，青春版用浅底与橙色小模块。

T 恤

胸前图形宽 90–120mm；背部可放大至 260mm。深浅两用时先保证蓝白单色可读。

贴纸

单枚保持 3–5mm 白边；组合拼版不超过 8 枚。使用数据流、坐标、SII 模块，文字短而准确。

徽章

圆形或异形均可；边缘安全区 ≥ 12%。避免校徽化中心对称，使用简洁蓝色符号系统。

笔记本

封面信息分三层：Logo、系列标题、抽象图形。科技版允许满版蓝，青春版以浅蓝灰留白为主。

水杯

杯身展开采用 180° 主视觉 + 180° 呼吸留白；线性图案优先，避免细小文字绕杯一圈。
