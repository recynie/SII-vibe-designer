# 系列口号与辅助文案 · v1

适用对象：上海创智学院文创产品两套系列。  
命名边界：仅使用已确认名称「上海创智学院」「SII」「Shanghai Innovation Institute」。  
禁用边界：不写校训、年份、院系、排名、导师姓名、未确认项目名称。

## 科技招生系列

调性：AI 前沿、专业可信、冷静科技。适合用于帆布袋、T 恤背印、笔记本封面、水杯杯身、徽章与贴纸组中的主标语。

### 01

**短口号**  
SII · Toward AI Frontiers

**辅助文字**  
上海创智学院面向人工智能前沿，以开放协作连接学习、研究与真实问题。

**适用建议**  
- 主推载体：T 恤背印、笔记本封面、帆布袋正面。
- 版式建议：英文短口号作为主标题，「SII」可放大；中文辅助文字小字号置于网格边缘。

### 02

**短口号**  
Build Intelligence with SII

**辅助文字**  
Shanghai Innovation Institute 聚焦 AI 创新语境，让想法进入实验、原型与应用。

**适用建议**  
- 主推载体：水杯环绕图形、贴纸组、徽章。
- 版式建议：搭配数据流、坐标网格或线性模块；避免写成校训式宣言。

### 03

**短口号**  
上海创智学院 · AI in Motion

**辅助文字**  
从问题出发，在 SII 的创客社区中推进探索、验证与迭代。

**适用建议**  
- 主推载体：帆布袋、笔记本封面、T 恤胸前小标。
- 版式建议：「上海创智学院」保持准确完整；「AI in Motion」可作为动态线条旁的辅助标题。

## 校园青春系列

调性：新生友好、清爽年轻、可日用。适合用于新生纪念语境的日常文创，但不出现未确认年份。

### 01

**短口号**  
Hello SII, Start Creating

**辅助文字**  
在上海创智学院，把第一天的好奇心带进每一次学习、实验与创作。

**适用建议**  
- 主推载体：帆布袋、贴纸组、笔记本内页贴标。
- 版式建议：浅底、品牌蓝与活力橙点缀；语气友好但不低龄卡通化。

### 02

**短口号**  
Daily Ideas at SII

**辅助文字**  
Shanghai Innovation Institute 的日常，从一个问题、一页笔记、一次动手开始。

**适用建议**  
- 主推载体：水杯、笔记本封面、贴纸组。
- 版式建议：适合手账式模块和小标签组合；辅助文字可拆成短行使用。

### 03

**短口号**  
Create, Learn, Repeat

**辅助文字**  
SII 的校园节奏，是持续学习、持续尝试，也持续把想法做出来。

**适用建议**  
- 主推载体：T 恤胸前/背部、徽章、帆布袋局部印花。
- 版式建议：三段式动词可与贴纸形状、路径线或轻量创客符号组合；不加入未确认组织信息。

## 统一使用提示

- 中文全称只写「上海创智学院」。
- 英文缩写只写「SII」。
- 英文全称只写「Shanghai Innovation Institute」。
- 产品小面积印刷时，优先使用短口号；辅助文字可作为吊牌、包装卡、笔记本封底或官网周边介绍文字。
