## 调性
- AI 前沿, 新生友好, 专业可信, 创客社区

## 视觉气质
- 冷静科技, 清爽年轻, 结构化, 可日用

## 系列方向
- 科技招生系列：以 #113E98 / #0168B7 为主，辅以深色或冷白底、网格、数据流、芯片线路、空间透视等抽象 AI 语言；突出“上海创智学院”“SII”与招生宣传的专业感。 [from-fact: 两个系列分别为科技风格和校园青春风格；科技风格偏招生宣传、品牌专业感，校园青春风格偏新生纪念、日常使用感]
- 校园青春系列：以 #F4F8FF 浅底、#113E98 品牌识别和 #F97316 活力强调形成差异；使用校园日常、创客贴纸感、手账式模块、轻量插画，但避免低龄卡通。 [from-fact: brief 情绪基调包含科技、品牌专业感、校园青春、日常使用感，且明确避免幼稚低龄化]

## 反 slop 禁区
- 紫渐变（除非 brief 明确要）
- emoji 图标
- 幼稚、低龄化卡通吉祥物或儿童玩具感
- 传统高校校徽式盾牌、橄榄枝、绶带、复杂纹章
- 不确定的校训、年份、院系、排名、导师姓名、项目名称
- 错写“上海创智学院”“SII”或混用其他 SII 机构含义
