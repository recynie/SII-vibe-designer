# 上海创智学院文创产品 设计交付

## brief 摘要
- 本次任务为上海创智学院设计一组面向新生与招生宣传的文创产品。
- 产品范围共 6 类：帆布袋、T 恤、贴纸、徽章、笔记本、水杯。
- 方案分为两条系列：科技招生系列与校园青春系列；前者偏专业可信与 AI 前沿气质，后者偏新生友好与日常使用感。
- 每类产品均需兼顾照片风格 mockup 展示与可印刷/延展的平面设计稿表达。
- 明确避免低龄化、传统高校校徽式视觉，以及任何未确认的校训、年份、院系等信息。

## 设计依据
- 色板依据 brand-spec：Primary `#113E98`、Secondary `#0168B7`、Background `#F4F8FF`、Ink `#232323`、Accent `#F97316`。
- 字体依据 brand-spec：Display `Montserrat-Medium`，Body `SourceHanSansCN-Regular`。
- 调性关键词：AI 前沿、新生友好、专业可信、创客社区。
- 视觉气质关键词：冷静科技、清爽年轻、结构化、可日用。

## 拒绝项
- 传统校徽/院徽重设计 | brief 明确避免传统高校校徽式设计，且本 run 聚焦文创产品而非重塑官方标志。
- 视频广告或动效物料 | brief 要求照片风格产品 mockup 与平面设计稿，视频/动效超出平面设计交付范围。
- 真实打样生产文件 | 本 run 只输出视觉 mockup、平面设计稿与规范页，不负责印厂刀模、材质采购或生产落地。

## 已知局限
- 科技招生系列中贴纸子项信息密度偏高，小字与局部细节清晰度弱于其它载体。
- 科技招生系列若进入进一步落地阶段，建议优先人工精修文字可读性与局部生成感。
- 校园青春系列整体通过，但贴纸与水杯的局部模块略密，若面向正式印刷可继续做减法优化。
- 印刷应用规范页经历一次字族硬门槛修复，最终通过版本为 `v2`，应以 `v2.html` / `v2.png` 为准。
